# Medical Insurance Prediction

## Project Overview
This project aims to develop a predictive model to estimate medical insurance costs based on various factors such as age, sex, BMI, number of children, smoking status, and region. The goal is to provide insights into how these factors influence insurance expenses, ultimately helping individuals to anticipate their healthcare costs.

## Instructions
1. **Clone the Repository**
   ```bash
   git clone https://github.com/ridhxm22/medical-insurance-prediction.git
   ```
2. **Navigate to the Project Directory**
   ```bash
   cd medical-insurance-prediction
   ```
3. **Install Dependencies**
   Ensure you have Python installed, then run:
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the Model**
   Execute the following command to start the analysis:
   ```bash
   python main.py
   ```
5. **Check the Results**
   After running the model, results will be displayed in the console or saved to specified output files.
